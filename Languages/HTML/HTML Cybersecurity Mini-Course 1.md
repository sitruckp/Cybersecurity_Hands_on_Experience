# HTML Cybersecurity Mini-Course Portfolio

## Overview
Learning HTML to create security awareness pages using Kali Linux and tools like Wireshark and Metasploit.

## Lesson Summaries
- **Lesson 1**: Built a basic security tip page with `<html>`, `<head>`, `<body>`, and `<p>`.
- **Lesson 2**: Added headings (`<h1>`) and multiple paragraphs.
- **Lesson 3**: Used attributes (`title`) for tooltips.
- **Lesson 4**: Created hyperlinks (`<a href>`) to resources.
- **Lesson 5**: Combined skills into a security dashboard.

## Career Prep Tips
- Use these pages in TryHackMe labs to simulate phishing awareness.
- Document skills for ISC2 CC and Google Cybersecurity certs.
- Explore CIS benchmarks for real-world security standards.
